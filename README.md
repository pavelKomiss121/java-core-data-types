
# Java: Практика работы с типами данных

## Цель проекта

Научиться:

- использовать примитивные и ссылочные типы данных в Java;
- понимать их поведение, ограничения и преобразования;
- замечать неочевидные особенности (переполнение, сравнение, точность и пр.);
- правильно применять арифметические и логические операции с учётом типов.

---

## Задание 1: Исследование границ типов данных

**Что нужно сделать:**

- Вывести `MIN_VALUE` и `MAX_VALUE` для всех числовых типов (`byte`, `short`, `int`, `long`, `float`, `double`, `char`);
- Показать переполнение (например: `byte 127 + 1 → -128`).

**Наблюдение:** Java не предупреждает о переполнении. Типы ведут себя циклически (например, byte → от 127 к -128).

---

## Задание 2: Международный конвертер валют

**Что нужно сделать:**

- Использовать `final` для курсов валют (`USD`, `EUR`, `CNY` к `RUB`);
- Рассчитать обмен между валютами;
- Показать проблему сравнения `double` (`0.1 + 0.2 == 0.3 → false`).

**Вывод:** тип `double` не точен. Для сравнения нужно использовать `Math.abs(a - b) < epsilon`.

---

##  Задание 3: Типы данных в действии

**Что нужно сделать:**

- Выполнить арифметику с разными типами: `int`, `double`, `char`, `boolean`;
- Показать деление, приведение типов, логические операции, поведение `char`.

##  Задание 4: Умный калькулятор

**Что нужно сделать:**

- Выполнить арифметику с правильным выбором типа;
- Обработать деление на 0, возведение в степень.

**Вывод:** результат зависит от типов: `double` даёт Infinity, `int` — ошибку.
